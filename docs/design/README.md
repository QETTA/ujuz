## 1. 개요
본 폴더는 UjuZ 디자인/UX 의사결정을 위한 리서치 바인더다.  
컨텍스트는 `어린이집 TO 알림`, `시설 비교`, `입학상담` 퍼널을 중심으로 구성되어 있다.

## 3. 권장 읽기 순서
1. `01` -> 원칙/용어 통일
2. `03` + `02` -> 권한/접근성 리스크 선제 정리
3. `05` + `06` -> 시장/사용자 질문 맥락 확보
4. `08` -> 인터뷰 가이드 확정
5. `07` -> 테스트 실행

## 4. 운영 규칙
1. 문서 수정 시 실제 화면 상태명(`P6/...`)과 동기화
2. 인터뷰/테스트 후 `06`, `08` 질문 우선 업데이트
3. 월 1회 `05` 경쟁사 표 업데이트(기능/퍼널 변동 반영)
