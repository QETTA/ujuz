# 프로젝트 방향 (v3, Phase 1~4 완료 후)

## 현재 위치
- **Phase 1~4 완료**: 67파일, +9,424줄, 프로덕션 MVP 기능 세트 구축
- TRL 7 (시스템 프로토타입 검증 완료)
- 10개 서브시스템 작동: 데이터파이프라인, 입소엔진, AI상담+가드레일, TO알림+Push/SMS, 커뮤니티, 결제/구독, 관리자, 모바일 6탭, 검색/비교, 데이터내보내기

## 핵심 가치
- **(a) 시설 탐색(지도) + (b) TO 알림 즉시성** = 제품 코어
- 네이티브 앱 = PMF에 직접 기여 (iOS 푸시 제약 해소)
- 웹 = 유입 채널 (SEO/결제), 모바일 = 핵심 경험

## 완료된 기능
- 입소 점수 계산 (Bayesian V2, LRU 캐시, 60+ 테스트)
- AI 스트리밍 채팅 (Claude, 가드레일, 사용량 제한)
- TO 감지 + Push/SMS/Email 알림
- Toss Payments 결제 + 구독 관리 (해지/환불/사용량)
- 시설 고급 검색 + 비교 (최대 3개)
- 아이 프로필 관리 (다자녀)
- 커뮤니티 (글쓰기/댓글, 웹+모바일)
- 관리자 대시보드 (통계, TO, 구독)
- 추천 히스토리 + 데이터 내보내기 (JSON/CSV)
- 모바일 6탭 (홈/검색/지도/채팅/알림/MY)
- 설정 (알림 토글, 방해금지, 계정 삭제)
